This workspace contains the example workflow from the GitHub Actions Quickstart guide.

- Workflow: `.github/workflows/github-actions-demo.yml`
- Trigger: `push` to any branch (the workflow runs on every push)

How to use:
1. Commit and push any change to this repository.
2. Open the **Actions** tab on GitHub and select **GitHub Actions Demo** to view runs and logs.

This README and the workflow were created from the GitHub Actions Quickstart documentation.
